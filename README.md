# Touche pas au Klaxon

Application de covoiturage développée en PHP (architecture MVC).

## Fonctionnalités

### Utilisateur
- Inscription / Connexion
- Créer un trajet
- Réserver un trajet
- Annuler une réservation

### Administrateur
- Lister les utilisateurs
- Supprimer un utilisateur
- Lister les agences
- Créer une agence
- Supprimer une agence
- Lister les trajets
- Supprimer un trajet
